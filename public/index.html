Funcionalidades

Sistema completo com backend real

Servidor Node.js

Backend real com Express.js, rotas de API REST e dados compartilhados entre todos os usuários.

Banco de dados SQLite

Dados reais persistidos em banco relacional. Funciona offline, sem necessidade de nuvem.

Senhas criptografadas

bcrypt com 10 rounds de salt. Nenhuma senha é armazenada em texto puro.

Autenticação JWT

JSON Web Token com expiração de 7 dias. Sessões seguras sem dependência de cookies.

Notificações reais

Quando o admin muda um status, o dono do item recebe notificação automaticamente no banco.

Exportar relatório CSV

Admin gera relatório real dos itens direto do servidor. Abre no Excel com um clique.

Como funciona

Simples, seguro e eficiente

1

Crie sua conta

E-mail @ulife.com.br e data de nascimento. Senha criptografada com bcrypt.

2

Registre ou busque

Todos os dados vão para o banco SQLite real. Qualquer usuário vê os mesmos itens.

3

Recupere seu objeto

Entre em contato e receba notificação quando o admin atualizar o status do seu item.
